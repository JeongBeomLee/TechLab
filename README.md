# KRAFTON Jungle Game TechLab 2기

크래프톤 정글 게임 테크랩 2기 프로젝트 모음입니다.

DirectX 11 기반의 게임 엔진을 처음부터 직접 구현하며, 언리얼 엔진의 핵심 구조를 학습하고 재현하는 과정을 담고 있습니다.

## Tech Stack

- **Language**: C++17
- **Graphics API**: DirectX 11
- **Physics**: NVIDIA PhysX
- **UI**: ImGui
- **IDE**: Visual Studio 2022
- **3D Assets**: FBX SDK

---

## Features

- Unreal Engine 스타일 Actor-Component 아키텍처
- 리플렉션 시스템 및 자동 직렬화
- Tile-Based Deferred Rendering
- GPU Skinning & Animation Blending
- PhysX 기반 물리 시뮬레이션
- Ragdoll & Cloth Simulation
- 파티클 시스템 및 이펙트
- 에디터 UI (기즈모, 프로퍼티 패널)
